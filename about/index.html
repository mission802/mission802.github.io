开源项目

  • gitee-exporter : Prometheus Gitee API exporter ，通过请求 Gitee API 获取项目 / 用户的相关数据（ star 数 / fork 数等）作为 Metrics 提供给 Prometheus 。
  • cmd-exporter : Prometheus CMD exporter ，使用 shell / 命令执行结果作为 Metrics 提供给 Prometheus 。
  • PrometheusStack : docker-compose 快速部署 Prometheus 相关组件（ Prometheus，Grafana，Consul，Nginx ）。
  • Prod-K8S-HA-Installer : Ansible + kubeadm 快速部署 K8S 高可用集群。
  • Earthling : docker-compose 快速部署 Elastic Stack相关组件（ Elasticsearch，Logstash，Kibana，Kafka，Nginx ）。
  • CI-Gitee : 基于码云的持续交付工作流
  • Autom : 运维自动化平台，服务端后端基于 Django + DRF 开发；服务端前端基于 vue-element-admin 开发；客户端基于 Go 的 gin 框架开发，支持采集节点信息并上报服务端、接收服务端派发的任务并执行，返回执行结果。包含 CMDB 、配置中心、工单系统、任务系统等功能模块。
  • CoreDNS : CoreDNS 二次开发项目，扩展 etcd 插件，支持顶级域名、二级、多级域名分别解析（支持多解析值负载）；解决 CNAME 解析时无法 fallthrough 外部 ZONE 问题。
